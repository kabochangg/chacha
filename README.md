# 勇者のあとしまつ

## ダンジョン清掃員の生活

`chacha` は、Webアプリゲーム「勇者のあとしまつ / ダンジョン清掃員の生活」を開発するためのプロジェクトです。

勇者が去ったあと、ダンジョンにはまだ仕事が残っている。プレイヤーは清掃員として、残骸、壊れた宝箱、スライムの跡、骨やがれきなどを片付け、素材を回収し、生活費を稼ぎながら道具を強化していきます。

## コンセプト

このゲームは、大規模なRPGではなく、短い清掃作業の気持ちよさを中心にした軽量な2Dアクションです。

優先する体験:

1. スマホで操作しやすいこと
2. 掃除アクションが気持ちいいこと
3. 素材回収と報酬獲得が分かりやすいこと
4. 1回3〜5分で終わること
5. 軽く動くこと

## MVP範囲

MVPでは次を実装対象にします。

- タイトル画面
- 拠点画面
- ダンジョン画面
- リザルト画面
- プレイヤー移動
- 掃除アクション
- アイテム回収
- HPまたはスタミナ
- 所持金
- 素材
- localStorageセーブ
- ほうき強化
- バッグ強化
- PWA
- Cloudflare Pages公開

MVPのダンジョンは「はじまりの地下道」1つに絞ります。強化対象は、清掃力に関わる「ほうき」と、所持容量に関わる「バッグ」を優先します。

## 対象外

MVPでは次を実装しません。

- サーバー
- DB
- ログイン
- ランキング
- 課金
- クラウドセーブ
- マルチプレイ
- チャット
- リアルタイム通信

旧「副業タイプ診断」サイト由来の広告、問い合わせ、AdSense、ASP運用もゲームMVPには持ち込みません。

## 操作方法

スマホ:

- 左下の固定仮想スティック: 移動
- 右下の大ボタン: 掃除
- 右下の小ボタン: 回避
- 対象に近づいた時の状況アクション: 宝箱、出口、素材、特殊オブジェクトを調べる
- 右上の一時停止ボタン: メニュー

PC:

- `W / A / S / D`: 移動
- `方向キー`: 移動
- `Space`: 掃除
- `Shift`: 回避
- `E`: 調べる
- `Esc`: メニュー

## ゲームループ

```txt
拠点で準備する
→ ダンジョンに入る
→ 残骸を掃除する
→ 素材を回収する
→ 出口から帰還する
→ リザルトで報酬を得る
→ ほうきやバッグを強化する
→ 次の清掃へ向かう
```

## 目標ビジュアル

メニュー画面とプレイ画面は、添付イメージを方向性として扱います。

- 暗い石畳のダンジョン背景
- ランタンの暖色光
- 黒革/木目/金縁のUI
- 上部リソースバー
- 下部タブ
- 左下仮想スティック
- 右下アクションボタン
- 右上ミニマップ風パネル

MVPでは画像そのものを素材として直貼りせず、Phaser描画と軽量なドット絵風素材で雰囲気を寄せます。

## 公開方針

初期公開先はCloudflare Pagesです。

- Viteの静的ビルド成果物を公開する
- PWAとしてホーム画面追加に対応する
- Service Workerで基本ファイルをキャッシュする
- スマホ縦画面を優先して確認する

## 受け入れ基準

MVPは次を満たしたら完成とします。

- タイトル画面からゲーム開始できる
- 拠点からダンジョンに入れる
- プレイヤーを操作できる
- 残骸を掃除できる
- 素材を拾える
- 出口から帰還できる
- リザルトが表示される
- 所持金が増える
- ほうきまたはバッグを強化できる
- ブラウザを閉じても続きから遊べる
- スマホで操作できる
- 移動しながら掃除ボタンを押し続けられる
- 仮想スティック操作中に画面スクロールが起きない
- Cloudflare Pagesで公開できる

最初の成功条件は、プレイヤーが「もう一回だけダンジョンに行きたい」と感じることです。
