# 勇者のあとしまつ

## ダンジョン清掃員の生活

`chacha` は、Webアプリゲーム「勇者のあとしまつ / ダンジョン清掃員の生活」を開発するプロジェクトです。

勇者が去ったあとのダンジョンには、残骸、壊れた宝箱、スライムの跡、がれきなどが残っています。プレイヤーは清掃員としてそれらを片付け、素材を回収し、報酬で道具を強化しながら次の清掃へ向かいます。

現在の目標は、MVPを磨き込み、v1公開版としてCloudflare Pagesで公開できる完成度にすることです。

## 体験の優先順位

1. スマホで操作しやすいこと
2. 掃除アクションが気持ちいいこと
3. 素材回収と報酬獲得が分かりやすいこと
4. 1回1〜3分で終わること
5. 軽く動くこと

豪華な画面、複雑な成長、オンライン要素、長いストーリーよりも、短い清掃ループの完成度を優先します。

## v1のゲーム内容

- ダンジョンは「はじまりの地下道」1つに絞る
- B1F〜B5Fを進行し、B5Fを最奥地にする
- 出撃ごとにマップ配置、開始位置、出口候補をある程度ランダムにする
- 各フロアに2〜5体の危険物を配置する
- 危険物は内部的な視線判定を持ち、発見時に `!` を出して追跡する
- 視線扇形の枠や範囲エフェクトは表示しない
- 瓦礫、木材、液体などの散乱物は掃除するまで通れない障害物として扱う
- 素材は近づくと自動回収する
- 帰還時に素材を換金するか、倉庫へ収納するか選べる
- 図鑑、持ち物、倉庫、依頼、ショップは最低限の機能を入れる
- localStorageで進行状況を保存する
- PWA対応し、Cloudflare Pagesで公開できる状態にする

## 操作

### スマホ

- 左下の固定仮想スティック: 移動
- 右下の大ボタン: 掃除
- 右下の小ボタン: 走る
- 対象に近づいた時のみ状況アクションを表示
- 右上の一時停止ボタン: メニュー
- 設定で左下スティック/右下ボタンと右下スティック/左下ボタンを切り替え可能

掃除はトグル操作です。一度押すと開始し、もう一度押す、対象から離れる、または掃除可能範囲を外れると中断します。

### PC

- `W / A / S / D`: 移動
- `方向キー`: 移動
- `Space`: 掃除
- `Shift`: 走る
- `E`: 状況アクション
- `J`: 清掃道具アクション
- `Esc`: メニュー

## 公開方針

初期公開先はCloudflare Pagesです。Viteの静的ビルド成果物を公開し、PWAとしてホーム画面追加と基本キャッシュに対応します。

## v1受け入れ基準

- タイトルからホーム、ダンジョン、リザルトへ進める
- 1回1〜3分で出撃が終わる
- B1F〜B5Fを進行できる
- 掃除、回収、帰還、報酬、強化が一通りつながる
- スマホ縦画面で移動しながら掃除できる
- 操作UI左右反転後も同時押しとスクロール抑止が崩れない
- ブラウザを閉じても続きから遊べる
- PWAとしてホーム画面に追加できる
- Cloudflare Pagesで公開できる

最初の成功条件は、プレイヤーが「もう一回だけダンジョンに行きたい」と感じることです。
