# 勇者のあとしまつ

## ダンジョン清掃員の生活

`chacha` は、Webアプリゲーム「勇者のあとしまつ / ダンジョン清掃員の生活」を開発するためのプロジェクトです。

勇者が去ったあと、ダンジョンにはまだ仕事が残っている。プレイヤーは清掃員として、残骸、壊れた宝箱、スライムの跡、骨やがれきなどを片付け、素材を回収し、生活費を稼ぎながら道具を強化していきます。

このリポジトリの次の目標は、MVPを磨き込み、v1公開版として「1〜3分で気持ちよく掃除して、もう一度潜りたくなる」完成度にすることです。

## 目指す体験

このゲームは大規模なRPGではなく、1〜3分で終わる清掃アクションと短い探索の気持ちよさを中心にした軽量な2Dゲームです。

優先する体験:

1. スマホで操作しやすいこと
2. 掃除アクションが気持ちいいこと
3. 素材回収と報酬獲得が分かりやすいこと
4. 1回1〜3分で終わること
5. 軽く動くこと

画面の豪華さ、複雑なシステム、オンライン機能、長いストーリーよりも、短いゲームループの完成度を優先します。

## 現在できること

- タイトル画面から開始できる
- 拠点から「はじまりの地下道」に入れる
- プレイヤーを移動できる
- B1F〜B5Fを進行でき、B5Fを最奥地として扱える
- 出撃ごとに開始位置、残骸配置、危険物数がある程度変わる
- 残骸に近づき、掃除ボタンを一度押すと掃除を開始できる
- 素材を自動回収できる
- バッグ容量は初期40を基準にし、階層が変わっても今回出撃中の収納数を引き継ぐ
- 危険物に触れるとダメージを受ける
- 危険物は内部的な視線判定、発見時の `!`、追跡を持つ
- ほうきや作業道具で危険物を追い払う清掃道具アクションがある
- 清掃率が上がると、ランダムな到達可能位置に出口が開く
- リザルトで報酬を確認し、回収素材を換金するか倉庫へ収納するか選べる
- ホームの下部タブと拠点導線で、持ち物、倉庫、マップ、依頼、ショップの最低限情報を確認できる
- 所持金でほうき、バッグ、追い払い、スタミナを強化できる
- 素材で強化ほうきや作業道具を生産できる
- 設定から操作UIを左右反転できる
- localStorageで続きから遊べる

## v1公開版で目指すこと

v1では、MVPの遊べる骨格を短い清掃バトル寄りの探索へ広げます。「はじまりの地下道」はB1F〜B5F構成にし、B5Fを最奥地にします。

- タイトル、ホーム、ダンジョン、リザルトの導線を完成形に近づける
- 添付イメージのような暗い石畳、ランタン光、金縁UI、下部タブの雰囲気に寄せる
- 各出撃ごとにマップ配置とスタート位置をある程度ランダムにする
- 各フロアのモンスター数は2〜5体の範囲でランダムにする
- モンスターは内部的な視線判定を持ち、プレイヤーに気づいたら `!` を出して追いかける。視線扇形の枠や範囲エフェクトは表示しない
- モンスターは壁を貫通できないが、壁や障害物に引っかかって停止し続けないようにする
- 瓦礫、木材、液体などの散乱物は通行判定を持ち、掃除しないと通れない障害物として扱う
- 条件達成後に出現する出口位置もランダム化する
- ダンジョンに行くごと、または深い階へ行くごとにモンスターを少し強くする
- 図鑑、持ち物、倉庫、依頼、ショップをv1相当の最低限機能として整理する。倉庫は下部タブを増やしすぎず、拠点または持ち物から入る導線にする
- モンスター/素材/宝の収集状況を図鑑で見られるようにする
- ほうき強化/生産は清掃速度や清掃範囲、バッグ強化は持ち帰り量、攻撃力/武器強化は危険物への対処力、スタミナ強化は行動継続力に直結させる
- 素材は売却だけでなく、ほうきや武器の生産/強化にも使う
- 設定から操作UIを左右反転し、左利き/右利きのどちらでも遊びやすくする
- PWAとしてホーム画面追加とCloudflare Pages公開に耐える状態にする

添付イメージは完成形の方向性として扱いますが、画像そのものをゲーム素材として直貼りする前提にはしません。Phaser描画、小さな生成テクスチャ、軽量なドット絵風パーツで雰囲気を寄せます。

## 操作方法

スマホ:

- 左下の固定仮想スティック: 移動
- 右下の大ボタン: 掃除、出口付近では帰還
- 右下の小ボタン: 走る
- 状況アクション: 宝箱、出口、特殊オブジェクトに近づいた時だけ表示する
- 素材: 近づいたら自動回収する
- 右上の一時停止ボタン: メニュー

初期設定は左下スティック/右下アクションです。設定から右下スティック/左下アクションへ反転でき、反転設定はlocalStorageに保存します。

PC:

- `W / A / S / D`: 移動
- `方向キー`: 移動
- `Space`: 掃除
- `Shift`: 走る
- `E`: 状況アクション
- `J`: ほうきで危険物を追い払う清掃道具アクション
- `Esc`: メニュー

`J` とスマホ上の追い払い操作は、清掃バトル寄りの補助操作です。主役は掃除と素材回収のままにしつつ、攻撃力や武器によって危険物への対処力が伸びるようにします。エフェクトは斧を横に振るような横薙ぎの弧を基本にし、討伐よりも作業道具で払う印象に寄せます。ボタン数は増やしすぎず、状況依存操作や掃除操作との統合も検討します。

## ゲームループ

```txt
ホームで準備する
→ ダンジョンに入る
→ B1F〜B5Fを短く探索する
→ 残骸を掃除する
→ 通路を塞ぐ瓦礫、木材、液体を片付けて道を開ける
→ 素材を回収する
→ 危険物を避ける、または追い払う
→ 出口から帰還する
→ リザルトで素材を換金するか倉庫へ収納する
→ 拠点で倉庫内素材を整理し、必要なら売る
→ 素材でほうきや武器を生産する
→ ほうき、バッグ、攻撃力、スタミナを強化する
→ 次の清掃へ向かう
```

## v1受け入れ基準

v1公開版は次を満たしたら完成とします。

- タイトルからホーム、ダンジョン、リザルトへ進める
- 1回1〜3分で出撃が終わる
- B1F〜B5Fを進行でき、B5Fが最奥地として扱われる
- 出撃ごとにマップ配置とスタート位置がある程度変わる
- 条件達成後の出口位置も到達可能な候補からランダムに変わる
- 各フロアのモンスター数が2〜5体の範囲で変わる
- スマホで移動しながら掃除を開始/中断できる
- 左右どちらの操作レイアウトでも仮想スティック操作中に画面スクロールが起きない
- 残骸を掃除できる
- 瓦礫、木材、液体などの散乱物は掃除するまで通れない
- 素材を自動回収できる
- バッグ容量が初期40で、階層移動後も今回出撃中の収納数が維持される
- 危険物を避ける、または清掃道具で追い払える
- モンスターの視線判定、発見時の `!`、追跡が機能し、視線扇形の枠や範囲エフェクトは表示されない
- モンスターが壁や障害物に引っかかって停止し続けない
- 出口から帰還できる
- リザルトで報酬、清掃率、素材、作業時間が分かり、素材を換金するか倉庫へ収納するか選べる
- 所持金が増える
- ほうき、バッグ、攻撃力、スタミナを強化できる
- 素材でほうきや武器を生産できる
- 図鑑、持ち物、倉庫、依頼、ショップがv1相当の最低限機能を持つ
- ブラウザを閉じても続きから遊べる
- PWAとしてホーム画面に追加できる
- Cloudflare Pagesで公開できる

最初の成功条件は、プレイヤーが「もう一回だけダンジョンに行きたい」と感じることです。
