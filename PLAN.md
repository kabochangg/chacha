# 勇者のあとしまつ 開発計画

## 方針

まずは大規模なRPGではなく、「掃除して、拾って、帰って、強化する」という小さなゲームループを完成させます。

優先することは次の5つです。

- スマホで操作しやすい
- 掃除アクションが気持ちいい
- 報酬獲得が分かりやすい
- 軽く動く
- 1回3〜5分で終わる

## Phase 1: 画面モック

目的は、雰囲気と画面構成を確認することです。

- Vite + TypeScript + Phaser.js のプロジェクト土台を作る
- タイトル画面を作る
- 拠点画面を作る
- ダンジョン画面の見た目を作る
- スマホ縦画面UIを作る
- 左下固定仮想スティックと右下2ボタンの配置を作る
- 右上に一時停止ボタンを置く
- 暗すぎないダンジョン、暖色系UI、軽量HD2D風の方向性を確認する

完了条件:

- タイトル、拠点、ダンジョンの画面遷移ができる
- スマホ縦画面で主要UIが見切れない
- 画面の雰囲気がゲームコンセプトに合っている
- 主要ボタンのタップ領域が48px以上、掃除ボタンが64px以上ある

## Phase 2: 操作プロトタイプ

目的は、アクションの操作感を確認することです。

- プレイヤー移動を実装する
- PCのWASD、方向キー操作を実装する
- スマホ向け固定仮想スティックを実装する
- 右下に掃除の大ボタンと回避の小ボタンを実装する
- 右上に一時停止ボタンを実装する
- カメラ追従を実装する
- 壁との当たり判定を実装する
- Space、Shift、E、Escの入力口を用意する
- タッチ操作中にブラウザスクロールが起きないようにする

完了条件:

- プレイヤーが移動できる
- 壁をすり抜けない
- スマホとPCの両方で移動できる
- 画面スクロールが誤作動しない
- 移動しながら掃除ボタンを押し続けられる

## Phase 3: 清掃アクション

目的は、ゲームの核になる掃除と回収を作ることです。

- 残骸を配置する
- 掃除ボタン長押しとSpaceキーで清掃できるようにする
- 残骸に耐久値を持たせる
- 清掃力に応じて耐久値を減らす
- 長押し中は進捗リング、短い揺れ、粉じん、掃き音で手応えを出す
- 長押しを離すと掃除を中断できるようにする
- 宝箱、出口、特殊オブジェクトの近くでは掃除ボタンを状況アクションへ切り替える
- 耐久値0で残骸を消す
- 素材をドロップする
- 素材は近づくと自動回収されるようにする
- 所持容量を実装する
- はぐれスライムなどの危険要素を最小限追加する

完了条件:

- 掃除アクションの結果が分かりやすい
- 素材を拾った感覚がある
- 所持容量を超えると拾えない
- 敵や罠が作業の邪魔として機能する
- 掃除、回避、状況アクションを押し間違えにくい

## Phase 4: ゲームループ

目的は、1回のプレイとして成立させることです。

- 拠点からダンジョン開始できるようにする
- 出口を配置する
- 清掃率80%以上、または出口撤退で帰還できるようにする
- HPが0になった場合の失敗処理を入れる
- リザルト画面を実装する
- 回収素材、売却見込み、獲得金額、被ダメージ、作業時間、ランクを表示する
- 拠点で素材売却できるようにする
- ほうきとバッグの強化を実装する

完了条件:

- 3分程度で1回のプレイが終わる
- 報酬を得た感覚がある
- 所持金が増える
- 強化によって次回プレイが少し楽になる
- もう一度潜る理由がある

## Phase 5: セーブ・PWA・公開

目的は、Webアプリとして遊べる状態にすることです。

- localStorage保存を実装する
- 続きから開始を実装する
- 設定情報を保存する
- manifest.jsonを追加する
- Service Workerを追加する
- アイコンとスプラッシュ表示を整える
- Cloudflare Pagesで公開できる構成にする
- iPhone、Android、PCで確認する

完了条件:

- ブラウザを閉じても続きから遊べる
- ホーム画面に追加できる
- オフライン起動の基本動作を確認できる
- Cloudflare Pagesで公開できる

## MVP完成条件

- タイトル画面からゲーム開始できる
- 拠点画面がある
- ダンジョンに入れる
- プレイヤーを操作できる
- 残骸を掃除できる
- 素材を拾える
- 出口から帰還できる
- リザルトが表示される
- 所持金が増える
- ほうきまたはバッグを強化できる
- セーブされる
- ブラウザを閉じても続きから遊べる
- スマホで操作できる
- Cloudflare Pagesで公開できる

## 受け入れテスト

基本動作:

- タイトル画面が表示される
- スタートボタンで拠点に遷移する
- 拠点からダンジョンに入れる
- プレイヤーが移動できる
- 壁をすり抜けない
- 掃除ボタンで残骸を消せる
- 素材を拾える
- 出口から帰還できる
- リザルトが表示される
- 所持金が増える
- 強化できる
- セーブできる
- 続きから再開できる

スマホ確認:

- iPhoneで画面が崩れない
- 縦画面で遊べる
- 仮想スティックが反応する
- 移動しながら掃除ボタンを押し続けられる
- ボタンが押しやすい
- 画面スクロールが誤作動しない
- 残骸、出口、素材の近くで状況アクションが分かりやすく切り替わる
- 3分程度のプレイで親指が疲れすぎない
- 音量設定が効く
- ホーム画面追加ができる

PC確認:

- キーボードで移動できる
- Spaceで掃除できる
- Escでメニューが開く
- ブラウザサイズ変更に耐える

## リスク対策

- 操作が複雑になりすぎないよう、MVPは移動、長押し掃除、回避、状況アクション、メニューに絞る
- HD2D風演出は軽量表現に留め、重い3Dや大量パーティクルは避ける
- 初期版ではサーバー、ログイン、ランキング、課金を入れない
- 1ダンジョン、強化2種類、敵2種類までに絞る
- まず遊べる状態を優先する
