# 実装メモ: 素材・モンスター・清掃ゲージ改善

## 今回直す不具合

- 2回目以降にダンジョンへ入った時、素材1つの回収で素材ゲージが大量に増えることがある。
  - 原因は、回収アニメ中の素材が物理判定に残り、`overlap` が複数フレーム発火するため。
  - 回収開始時点で `collected` を立て、物理判定を無効化する。

- モンスターに触れてもダメージが入らないことがある。
  - 矩形判定だけに頼らず、中心距離でも接触判定する。
  - 連続ヒットしすぎないよう、短い無敵時間は維持する。

## 追加仕様

- 一時停止メニューから、今回ダンジョン内で持っている素材、バッグ容量、清掃率、HP、所持金を確認できるようにする。
- 右下に攻撃ボタンを追加する。PC操作では `J` キーを攻撃にする。
- 攻撃は前方の短いほうきスイングとして扱い、モンスターを数回で倒せるようにする。
- 倒したモンスターは素材を1つ落とす。
- モンスターは単純な反復ではなく、`sin/cos` を使った規則的な2軸移動にする。
- がれきは大きさごとに清掃率の増加量を変える。
  - 小さいがれき: 5%
  - 中くらいのがれき: 10%
  - 大きいがれき: 15%
- 掃除途中で掃除ボタンを離すと、未完了がれきの個別掃除ゲージが少しずつ戻る。

## 確認項目

- `npm.cmd run build` が成功する。
- 素材を1つ拾った時、素材数が1だけ増える。
- メニューで今回の素材数が確認できる。
- モンスター接触でHPが減る。
- 攻撃ボタンまたは `J` キーでモンスターを倒せる。
- 小/中/大がれきの清掃完了で清掃率が5/10/15%増える。
- 掃除を中断すると、掃除中だったがれきの進捗が徐々に戻る。
