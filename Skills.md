# Skills

## このプロジェクトで必要なスキル

「勇者のあとしまつ / ダンジョン清掃員の生活」を公開するために必要な開発スキルをまとめます。

最初から大きなRPGを作り込まず、MVPの短い清掃ループを完成させながら段階的に伸ばします。

## Phaser.js / TypeScript

- PhaserのScene、GameObject、Physics、Camera、Inputを理解する
- `BootScene`、`TitleScene`、`BaseScene`、`DungeonScene`、`ResultScene` の役割を分ける
- TypeScriptでプレイヤー、残骸、素材、強化、セーブデータの型を定義する
- ゲームロジックと表示処理を混ぜすぎない
- 毎フレームの重い処理を避け、スマホでも軽く動かす

## 2Dゲーム設計

- 見下ろし型の小さなダンジョンマップを作る
- 壁、床、出口、残骸、素材、危険物を配置する
- プレイヤー移動、当たり判定、カメラ追従を調整する
- 敵は討伐対象ではなく、作業を邪魔する危険要素として扱う
- 1回3〜5分で終わるテンポを保つ

## スマホUI / 入力設計

- スマホ縦画面を最優先にする
- 左下に固定仮想スティックを置く
- 右下に掃除の大ボタンと回避の小ボタンを置く
- 状況アクションは常時ボタンにせず、必要な時だけ掃除ボタンを切り替える
- 素材は近づいたら自動回収する
- タップ領域を広めに取り、押し間違いを減らす
- 主要ボタンは64px以上、その他は48px以上を目安にする
- タッチ操作中のブラウザスクロールを防ぐ
- セーフエリアを考慮し、主要操作を画面端へ寄せすぎない
- PCではWASD、方向キー、Space、Shift、E、Escを使えるようにする

## 掃除アクション設計

- 掃除は長押し操作にする
- 長押し中は進捗リング、短い揺れ、粉じん、掃き音で手応えを出す
- 長押しを離したら掃除を中断する
- 残骸の耐久値が0になったら素材をドロップする
- 耐久値が低い残骸はすぐ消し、耐久値が高い残骸は複数拍かけて消す
- 対象選択は、プレイヤーが向いている方向または最寄りの残骸を優先する

## UIアート / 演出

- 添付画像の方向性に合わせ、暗い石畳、暖色のランタン光、金縁UIを基調にする
- メニュー画面は上部リソースバー、中央メニュー、下部タブを持たせる
- プレイ画面は上部HUD、右上ミニマップ風パネル、左下スティック、右下アクションボタンを持たせる
- 背景はタイルベースで軽量に作る
- 光、影、粉じん、短い揺れは控えめに使う
- 画像や音声はファイルサイズを抑える
- 仮素材でも、用途と違いが分かる名前にする

## localStorageセーブ

- 所持金、所持素材、ほうきレベル、バッグレベル、進行状況、設定を保存する
- セーブデータには `version` を含める
- 壊れたセーブデータを読んでもゲームが停止しないようにする
- 保存形式は将来のDB化を見越して単純なJSONにする

## PWA / Webアプリ化

- `manifest.json` を整える
- アイコン、テーマカラー、画面向き、表示モードを設定する
- Service Workerで基本ファイルをキャッシュする
- ホーム画面追加とオフライン起動の基本動作を確認する
- スマホブラウザの音声自動再生制限を考慮する

## Cloudflare Pages

- Viteの静的ビルド成果物をCloudflare Pagesで公開できるようにする
- MVPではサーバー、DB、認証、ランキングを使わない
- 将来サーバーが必要になった場合は、Cloudflare Workers、Supabase、Firebase、Cloudflare D1などを検討する
- 公開前にスマホ実機、PC、ブラウザサイズ変更を確認する

## ゲームバランス

- ほうき強化は清掃速度に直結させる
- バッグ強化は持ち帰れる素材数に直結させる
- 報酬は分かりやすく、次の強化目標が見えるようにする
- 危険物は強くしすぎず、作業中の緊張感を作る程度にする
- 「もう一回だけダンジョンに行きたい」と感じる短いループを優先する

## テスト観点

- タイトルから拠点、ダンジョン、リザルトへ進めるか
- スマホで移動と掃除がしやすいか
- 移動しながら掃除ボタンを押し続けられるか
- 掃除、回避、状況アクションを押し間違えにくいか
- PCキーボードで操作できるか
- 壁をすり抜けないか
- 残骸を掃除して素材を拾えるか
- 出口から帰還できるか
- 所持金と素材が増えるか
- 強化できるか
- セーブと続きから開始が動くか
- ビルド後もCloudflare Pagesで公開できるか
